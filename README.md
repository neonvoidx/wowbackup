# World of Warcraft Addon Backup
- contains all my addons
- contains all addon settings via WTF folder
  - If you want to copy, you need to go into WTF folder and change your realm/char names appropriately
- contains my Fonts
- I've also added profile import codes for BetterBlizzPlates, BetterBlizzFrames, SArena Reloaded, Blizzard edit mode, and Sensei
  - BBP_profile.txt
  - BBF_profile.txt
  - Sensei_profile.txt
  - EditMode_profile.txt
  - SArena_profile.txt
- For upscaled Icon pack use [this](https://github.com/AcidWeb/Clean-Icons-Mechagnome-Edition/releases/latest), and you place them under `Wow folder/_retail_/Interface`, so you should after copying have `Wow folder/_retail_/Interface/ICONS`
